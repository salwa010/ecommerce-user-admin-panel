# Angular E-Commerce Platform (User & Admin Panel)

A scalable full-stack-style eCommerce web application built using **Angular** and **TypeScript**, featuring separate **User** and **Admin** modules with a clean, modular architecture.

---

## 🚀 Features

### 👤 User Side

* Product listing with dynamic data rendering
* Product details page
* Add to cart / remove from cart
* Order placement flow
* Responsive UI for all devices

### 🛠 Admin Panel

* Product management (add, edit, delete)
* Category management
* Order tracking and management
* Dashboard-style data handling

---

## 🧠 Key Highlights

* Angular component-based architecture
* Reusable and modular components
* Lazy loading for performance optimization
* Reactive forms for data handling
* Routing for user/admin separation
* API integration for dynamic data
* Responsive UI design (mobile-friendly)

---

## 🛠 Tech Stack

* Angular
* TypeScript
* HTML5
* SCSS
* RxJS
* REST APIs

---

## 📁 Project Structure

```
src/

 │   ├── admin/       # Admin module
 │   ├── shared/      # Shared components/services
 │   ├── core/        # Core services/guards
 │   └── app-routing
```
